# Launcharr Release Announcement

## Headline

Launcharr `v0.2.44` is now available

## Short Summary

Launcharr `v0.2.44` delivers a new Launcharr deployment-summary widget, stronger app/widget status reliability, and quality/security fixes across Immich, qBittorrent, and settings flows.

## What's New

- Added a `Launcharr` system widget that shows online/offline/total app counts with a color-coded app-up percentage.
- Added deployment-summary widget configuration controls for metric columns and stat visibility.
- Added built-in `QNAP` app + widget stats adapter (`CPU`, `Memory`, `Volume`) and icon coverage.
- Added API-key reveal/hide toggle UX with wider API-key input fields across settings forms.
- Added setup and Plex PIN-status rate limits, and tightened Plex PIN session handling.
- Fixed custom icon upload visibility/path issues so uploaded app/system icons now appear and can be selected in settings.

## Why It Matters

- Operators get an at-a-glance service health summary directly in widget bars.
- Widget/app status cards are more accurate for authenticated upstreams (qBittorrent) and widget-only integrations (QNAP).
- Media overview reliability improves for Immich recent-photo rendering across API variants.
- Custom branding/icon workflows in settings are reliable again for ongoing admin customization.

## Upgrade Notes

- Docker image: `mickygx/launcharr:v0.2.44`
