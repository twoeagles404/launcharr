# Launcharr Release Announcement

## Headline

Launcharr `v0.2.46` is now available

## Short Summary

Launcharr `v0.2.46` adds built-in Curatorr support, app-specific sidebar launch sections, and embedded theme syncing for Curatorr iframe launches.

## What's New

- Added built-in `Curatorr` support with dedicated icons and sidebar launch sections for its main areas.
- Added role-aware app submenu links so Launcharr can route users straight into app-specific launch destinations.
- Updated Curatorr iframe launches to inherit Launcharr theme and square-corner preferences and stay synced after UI changes.

## Why It Matters

- Admins can expose Curatorr as a first-class built-in app instead of maintaining a custom launcher entry.
- Users can jump directly into the right Curatorr section from the Launcharr sidebar instead of landing on a generic launch page.
- Embedded Curatorr sessions now stay visually aligned with Launcharr theme settings.

## Upgrade Notes

- Docker image: `mickygx/launcharr:v0.2.46`
- Supplementary tags: `mickygx/launcharr:development`, `mickygx/launcharr:latest`
- No migration steps required.
- Rebuild/restart Launcharr and hard refresh browser assets once after upgrade.
